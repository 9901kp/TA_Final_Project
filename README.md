## Project Overview
This project is a comprehensive **End-to-End Test Automation Framework** developed for the **Automation Exercise** platform.  
It covers both **UI (Frontend)** and **API (Backend)** testing using modern automation tools and best practices.

- **Target Application (UI):** https://automationexercise.com  
- **UI Test Cases Reference:** https://automationexercise.com/test_cases  
- **API Reference:** https://automationexercise.com/api_list

## Automation Stack
The project is implemented using the following technologies (as required):

- **Java**
- **Selenium WebDriver**
- **TestNG**
- **RestAssured**
- **Maven**
- **Allure Reporting**
- **Page Object Model (POM)**
